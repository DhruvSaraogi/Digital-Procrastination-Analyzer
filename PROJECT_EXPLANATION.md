    user = models.ForeignKey(User, ...)           # Link to user
    subject = models.CharField(max_length=200)    # Study subject
    start_time = models.DateTimeField()           # When study started
    end_time = models.DateTimeField()             # When study ended
    focus_score = models.IntegerField(...)        # 1-10 rating
    notes = models.TextField(blank=True)          # Optional notes
    created_at = models.DateTimeField(auto_now_add=True)
    
    @property
    def duration_minutes(self):                   # Calculate study duration
        delta = self.end_time - self.start_time

- **@property:** Methods accessible like attributes (e.g., `session.duration_minutes`)
- **auto_now_add:** Automatically set when record created

### 5. **ProductivityMetrics Model** - Daily summary statistics

```python
class ProductivityMetrics(models.Model):
    user = models.ForeignKey(User, ...)
    date = models.DateField()
    total_study_minutes = models.FloatField(default=0)
    avg_focus_score = models.FloatField(default=0)
    total_distraction_minutes = models.FloatField(default=0)
    distraction_ratio = models.FloatField(default=0)  # distraction/study
    sessions_count = models.PositiveIntegerField(default=0)
    tasks_completed = models.PositiveIntegerField(default=0)
    habits_completed = models.PositiveIntegerField(default=0)
    productivity_score = models.FloatField(default=0)  # Overall score 0-100
    
    class Meta:
        unique_together = ['user', 'date']  # Only one record per user per day
```

- **Aggregated daily data:** Computed from study sessions, distractions, tasks, habits
- **productivity_score:** Main KPI (0-100)

**Productivity Score Calculation:**
```
formula:
  focus_component = (avg_focus_score / 10) * 40%        # Focus contributes 40%
  study_component = min(total_study / 240 hours, 1) * 30%  # Study time 30%
  distraction_penalty = distraction_ratio * 20%         # Distractions -20% max
  task_bonus = min(tasks_completed / 3, 1) * 10%       # Tasks +10%
  
  score = focus_component + study_component - distraction_penalty + task_bonus
  final = max(0, min(score, 100))
```

### 4. **Habits Management**

#### `complete_habit(request, pk)` - Mark habit done (AJAX)

```python
@login_required
def complete_habit(request, pk):
    """Mark a habit as completed for today (AJAX)."""
    habit = get_object_or_404(Habit, pk=pk, user=request.user)
    today = timezone.now().date()
    
    if habit.last_completed == today:
        return JsonResponse({'status': 'already_done'})
    
    # Check if yesterday → maintain streak
    if habit.last_completed == today - timedelta(days=1):
        habit.current_streak += 1
    else:
        habit.current_streak = 1  # Start new streak
    
    habit.longest_streak = max(habit.longest_streak, habit.current_streak)
    habit.total_completions += 1
    habit.last_completed = today
    habit.save()
    
    return JsonResponse({
        'status': 'success',
        'streak': habit.current_streak,
        'total': habit.total_completions,
        'stability': habit.stability_score,
    })
```

**Streak Logic:**
```
Yesterday: completed → streak +1
Gap: > 1 day → streak = 1 (restart)
Today: already done → no change
```

### `compute_daily_metrics(user, date)` - Calculate daily score

```python
def compute_daily_metrics(user, date=None):
    if date is None:
        date = timezone.now().date()
    
    # Fetch all day's data
    sessions = StudySession.objects.filter(user=user, start_time__date=date)
    distractions = DistractionLog.objects.filter(user=user, timestamp__date=date)
    
    # Calculate components
    total_study = sum(s.duration_minutes for s in sessions)
    avg_focus = sessions.aggregate(avg=Avg('focus_score'))['avg'] or 0
    total_distraction = distractions.aggregate(...)['total'] or 0
    distraction_ratio = (total_distraction / total_study) if total_study > 0 else 0
    
    # Productivity score formula (0-100)
    focus_component = (avg_focus / 10) * 40        # Focus: 40% weight
    study_component = min(total_study / 240, 1) * 30  # Study time: 30% (4h = 100%)
    distraction_penalty = min(distraction_ratio, 1) * 20  # Distractions: -20% max
    task_bonus = min(tasks_done / 3, 1) * 10      # Tasks: +10% (3 tasks = 100%)
    
    productivity_score = max(0, 
        focus_component + study_component - distraction_penalty + task_bonus
    )
    
    # Update or create metrics record
    metrics, _ = ProductivityMetrics.objects.update_or_create(
        user=user,
        date=date,
        defaults={
            'total_study_minutes': total_study,
            'avg_focus_score': round(avg_focus, 2),
            'total_distraction_minutes': total_distraction,
            'distraction_ratio': round(distraction_ratio, 4),
            'sessions_count': sessions.count(),
            'tasks_completed': tasks_done,
            'habits_completed': habits_done,
            'productivity_score': round(productivity_score, 2),
        }
    )
```

```python
def detect_patterns(user):
    """Detect recurring patterns over 30-day period."""
    month_ago = timezone.now() - timedelta(days=30)
    sessions = StudySession.objects.filter(user=user, start_time__gte=month_ago)
    
    # Group by time of day
    time_blocks = {
        'Morning (6 AM - 12 PM)': sessions.filter(start_time__hour__lt=12),
        'Afternoon (12 PM - 5 PM)': sessions.filter(start_time__hour__gte=12, start_time__hour__lt=17),
        'Evening (5 PM onwards)': sessions.filter(start_time__hour__gte=17),
    }
    
    for label, qs in time_blocks.items():
        if qs.count() >= 3:
            avg = qs.aggregate(avg=Avg('focus_score'))['avg'] or 0
            
            if avg >= 7:  # High focus
                BehaviorPattern.objects.update_or_create(
                    user=user,
                    pattern_type='peak_focus',
                    defaults={
                        'description': f"Peak focus during {label} ({avg:.1f}/10)",
                        'confidence': min(avg / 10, 1.0),
                        'data_points': qs.count(),
                    }
                )
            elif avg <= 4:  # Low focus
                BehaviorPattern.objects.update_or_create(
                    user=user,
                    pattern_type='low_focus',
                    defaults={
                        'description': f"Low focus in {label} ({avg:.1f}/10)",
                        'confidence': min((10 - avg) / 10, 1.0),
                        'data_points': qs.count(),
                    }
                )
```

def generate_alerts(user):
    """Generate various alert types based on conditions."""
    now = timezone.now()
    
    # Alert 1: Streak milestones
    habits = Habit.objects.filter(user=user, is_active=True)
    for habit in habits:
        if habit.current_streak == 7:  # 1 week milestone
            Alert.objects.create(
                user=user,
                alert_type='streak_milestone',
                severity='success',
                title=f"1-Week Streak! 🎉",
                message=f"'{habit.name}' is now a habit with 7 days!"
            )
        elif habit.current_streak == 30:  # 1 month
            Alert.objects.create(...)
    
    # Alert 2: Overdue tasks
    for task in Task.objects.filter(user=user, status!='completed'):
        if task.deadline and task.is_overdue:
            Alert.objects.create(
                user=user,
                alert_type='task_overdue',
                severity='danger',
                title='Task Overdue!',
                message=f"'{task.title}' was due {(now - task.deadline).days} days ago"
            )
    
    # Alert 3: Low focus warning
    today_metrics = ProductivityMetrics.objects.filter(user=user, date=now.date()).first()
    if today_metrics and today_metrics.avg_focus_score < 3:
        Alert.objects.create(
            user=user,
            alert_type='low_focus',
            severity='warning',
            title='Low Focus Today',
            message='Your focus score is very low. Take a break and refresh!'
        )

## 🚀 KEY TECHNICAL FEATURES

### 1. **Streak Logic**
```
- If completed yesterday → streak += 1
- Gap > 1 day → streak = 1 (restart)
- Already done today → no change
- Track longest ever streaks
```

### 2. **Productivity Score**
```
Formula combines:
- Focus quality (40%)
- Study duration (30%)
- Distraction penalty (-20%)
- Task completion bonus (+10%)
= 0-100 score
```

### 3. **Rule-Based Suggestions**
15 different rules analyze:
- Study behavior (sessions, focus)
- Distraction patterns (types, frequency)
- Habit consistency (streaks, frequency)
- Task management (overdue, volume)
- Time preferences (peak focus hours)
- Progress trends (improvement detected)

### 4. **Pattern Detection**
- Analyzes 30 days of data
- Identifies peak focus times
- Detects distraction triggers
- Confidence scores (0-1)
- Data point count

### 5. **Alert System**
- Streak milestones (7, 30 days)
- Low focus warnings
- Overdue task notifications
- Improvement celebrations
- Dismissible and read-tracking

### 6. **AJAX Interactions**
- Mark habit done → updates streak without page reload
- Mark task complete → updates status instantly
- Rate suggestion → saves feedback
- Dismiss alert → removes notification

## 🔐 SECURITY FEATURES

1. **@login_required decorator**
   - Blocks unauthenticated access
   - Redirects to login page

2. **User isolation**
   - get_object_or_404(Model, ..., user=request.user)
   - Prevents users from accessing others' data

3. **CSRF protection**
   - {% csrf_token %} in all forms
   - Django middleware validates tokens

4. **Password hashing**
   - django.contrib.auth.hashers
   - Passwords stored as hash, never plain text

5. **SQL injection prevention**
   - ORM validates all queries
   - No raw SQL in views
